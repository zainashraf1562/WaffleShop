The Waffle Shop application allows users to order waffles, milkshakes, and ice cream using the menu option. 
This application is meant for customers to build their own waffles or select from already-made signature waffles, add toppings, and get their total cost. 
When the order is complete, this application saves a receipt with all items ordered. 
This application is designed to make ordering from the Waffle Shop easy and keep track of what one bought.

The Main function of this application:

- Start a new order
- Add items to your order
    - Add a custom waffle
    - Add a signature waffle
    - Add milkShakes
    - Add Ice Cream
- Customize a waffle
    - Choose size
    - Choose waffle Type
    - Add toppings
    - Add extra toppings
    - Remove toppings
- View your current order
- See the total price
- Checkout and create a receipt
    - Receipt includes all items
    - Shows total cost
    - Saves receipt with date and time
- Cancel an order

## User Stories

- As a user, I would like to create my own waffle, being able to choose the size, waffle type, and toppings so that I can customize my order exactly the way I like it.
- As a user, I want to add icecream and milkshakes to my order so I can complete my meal on one check out.
- As a user, I would like to see an overview of my order with the item and prices listed so that I can confirm everything before purchase.
- As a user, I would like to see the total cost of my order so that I know how much to pay before checkout.
- As a user, I want to add premium toppings, such as additional chocolate or meat, so I can upgrade my waffle by adding flavor for an extra cost.
- As a user, I want the system to save my order details into a receipt so that I can have a record of what I purchased.

### Running the Application in IntelliJ

Follow these steps to get your application running within IntelliJ IDEA:

1. Open IntelliJ IDEA.
2. Select "Open" and navigate to the directory where you cloned or downloaded the project.
3. After the project opens, wait for IntelliJ to index the files and set up the project.
4. Find the main class with the `public static void main(String[] args)` method.
5. Right-click on the file and select 'Run 'Main.java' to start the application.
